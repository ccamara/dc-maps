This module provides a light-weight alternative to marker popups for Leaflet maps using labels, provided
by the Leaflet label plugin (https://github.com/Leaflet/Leaflet.label).

Installation
------------

1. Make sure you have the Leaflet module enabled.

2. Download the Leaflet Label library from https://github.com/Leaflet/Leaflet.label and put
   it in your libraries folder renamed to "leaflet_label".

3. In your Leaflet Views, select a title field. This field will now be rendered as a label.
